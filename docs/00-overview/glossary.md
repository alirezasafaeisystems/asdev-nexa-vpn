# Glossary (واژه‌نامه)

---

## اصطلاحات فنی

| اصطلاح | تعریف |
|--------|-------|
| **VLESS** | پروتکل VPN مدرن با رمزنگاری سبک |
| **Hiddify** | پنل مدیریت VPN متن‌باز |
| **Config** | فایل پیکربندی اتصال VPN |
| **UUID** | شناسه یکتا برای هر کاربر VPN |
| **Idempotency** | ویژگی عملیات که با تکرار، نتیجه یکسان می‌دهد |

---

## اصطلاحات بیزینس

| اصطلاح | تعریف |
|--------|-------|
| **Subscription** | اشتراک فعال کاربر |
| **Invoice** | فاکتور پرداخت |
| **Plan** | پلن اشتراک (Basic, Pro, Premium) |
| **Trial** | دوره آزمایشی رایگان |
| **Traffic** | حجم مصرفی (گیگابایت) |

---

## وضعیت‌ها

### Invoice Status
| وضعیت | توضیح |
|-------|-------|
| CREATED | فاکتور ایجاد شده |
| PENDING | در انتظار پرداخت |
| PAID | پرداخت شده |
| EXPIRED | منقضی شده |
| REFUNDED | مسترد شده |

### Subscription Status
| وضعیت | توضیح |
|-------|-------|
| ACTIVE | فعال |
| EXPIRED | منقضی |
| CANCELED | لغو شده |
| SUSPENDED | معلق |

### Server Status
| وضعیت | توضیح |
|-------|-------|
| ONLINE | آنلاین |
| OFFLINE | آفلاین |
| MAINTENANCE | در حال تعمیر |
